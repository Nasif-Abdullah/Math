As it will be discussed gradually, real numbers are of great importance, as they are continuous and ordered, they can be used as means to measure distances, they have no "holes" so any convergent series in it converges to a real number, and many other properties. Classical analysis begins with the $$\sqrt{2}$$ problem, so we discuss it now.

$$\sqrt{2}$$ is the solution to the equation, $$q^2 = 2$$. Notice the following sequence- $$\{1, 1.4, 1.41, 1.414, 1.4142,...\}$$ 

The sequence is built on rational numbers, which we have constructed, recall, rational numbers $$\mathbb{Q}$$ are numbers that can be represented by $$\frac{m}{n}\ \ (m,n \in \mathbb{Z})$$. For example, $$1.41 = \frac{141}{100}$$.

The sequence converges to $$\sqrt{2}$$ but, as we haven't constructed irrational numbers, we can't understand to which object this sequence converges to.

We set $$A$$ to be the set of all $$p^2<2$$ and $$B$$ to be the set such that $$p^2>2$$, where $$p,q \in \mathbb{Q}$$. For example- consider the following equation- 

$q = p - \frac{p^2 -2}{p+2} = \frac{2p + 2}{p+2}$ (i)

$q^2 - 2 = \frac{2(p^2 - 2)}{(p+2)^2}$ (ii- By simple algebraic manipulation)

Now, set any $$p \in A$$, i.e. $$(p^2 < 2) \rightarrow (p^2 -2 <0)$$. We have from equation (i), that $$q > p$$ and from equation (ii) we have, $$q^2 -2 < 0$$, so $$q \in A$$, similar for B can be shown, and it implies that the set A and B has no largest or smallest numbers, more accurately, we have shown we can find a $$q \in A$$ such that $$(\forall p \in A) \ \ (q>p) \wedge (q \in A)$$, and similar for B . If A had a largest number, and B had a smallest number, say s, then we could've written $$s^2 = 2$$.

As $$p,q$$ both are rational numbers, and we are bound in a rational "field", we can say $$\sqrt{2} \notin \mathbb{q}$$. 



$\blacksquare$
